Fixed Scroll Parallax imitation

The Markup

Create three parent containers which will act as the parallax-like scroll context i.e. the images. Within each of the three parent containers, we add our header or title that overlies the image. We will center this text vertically and horizontally. Add separate containers after each of the three parent containers for the copy text and this allows us to scroll the page down as well as breaking up the monotony of the page.

CSS

Use background properties to place the image, min-heights, position it, no repeat, cover the wrapper etc. Add colour, typeface, sizes etc. to contrast sections. Adjust settings to suit your images.

Section Three

We use the CSS property background-attachment: fixed; which means the background image remains fixed in position while the other content scrolls down the page. The default is 'scroll' which means both the image and content will be scrolled out of view proportionally down the page. Although this gives the appearance of delayed scrolling, it is not quite a parallax effect. This is when parts of the website scroll or move a different pace from other parts often used to differentiate between background and foreground. It is an optical illusion that gives a sense of depth of view. It is advised to turn off the parallax effect on mobile.
