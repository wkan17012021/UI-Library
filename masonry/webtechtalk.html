A CSS masonry based off WebTech Talk

Video URL

Uses columns and break-inside properties to achieve specific custom masonry layout (finite number of cards). The overlap between columns is controlled by content and to achieve responsiveness, apply custom media queries.

As of 2022, masonry can not be achieved with CSS alone if you want the layout to auto-fill when you have new content and be responsive. It requires a javascript library. There is an experimental property grid-template-rows: masonrycoupled with display grid and template columns. But, it has no support on browser yet.
